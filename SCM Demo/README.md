# Meridian Pharma Simulation - The Chain

This project is a React-based supply chain management simulation for postgraduate education.

## Installation and Running

To run this project locally with a proper build system (Node.js/NPM):

1.  **Install Node.js**: Ensure you have Node.js (version 16+) installed. You can download it at [nodejs.org](https://nodejs.org).
2.  **Install Dependencies**: Run the following command in this directory:
    ```bash
    npm install
    ```
3.  **Run Development Server**: Start the local server with:
    ```bash
    npm run dev
    ```
    This will open the site in your browser with hot-reloading.

## Standalone Version
The `the-chain-v3.html` file (and upcoming `index.html`) can also be opened directly in a browser. It uses Tailwind CSS via CDN and React via CDNs.

The application has been updated to adhere to:
